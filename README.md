## Getting started

These instructions will get a copy of the project up and running on your local machine for development and testing.

Prerequisites
- Node.js (>= 14) and npm or yarn installed

Clone the repo
git clone https://github.com/rshabhkulkarnii/React-Project.git
cd React-Project

Install dependencie
npm install
# or
yarn install

Start development server
npm start
# or
yarn start

Open http://localhost:3000 (or the port printed in the console) to view the app.

Build for production
npm run build
# or
yarn build


Run tests
npm test
# or
yarn test
